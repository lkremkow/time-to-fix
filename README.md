# Time to Fix

  Community site to calculate vulnerability time to fix statistics.


## Requirements

   * [Node.js](https://nodejs.org/en/); tested with 8.9.4 and 8.11.3 LTS
   * [NPM](https://www.npmjs.com); tested with 5.6.0, 6.1.0, and 6.3.0
