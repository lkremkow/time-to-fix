About Time to Fix

How Does This Work

When you login with your account credentials, this site will call their API and extract the list of vulnerabilites present. A condensed version is temporarily stored on our server while we wait for the download from the platform to complete. When all the averages are calculated, the data is deleted.

Definitions

average time it takes to fix
Of all the vulnerabilities found in a given perimeter, we consider all the vulnerabiliteis that are marked as "Fixed" by the scanner. We calculate the period of time between the first detection and the last detection. This period is conisdered the time to fix.
average number of vulnerabilities per host
We consider the total numnber of vulnerabilities detected for a given host as per vulnerability scanner and calculate an average across all hosts in a given scope.
vulnerabilities
Confirmed vulnerabilites (red) levels 3, 4, or 5 as detected by the Vulnerability Management or Cloud Agent. Information Gathered (blue) and Potential Vulnerabiliteis (yellow) are ignored.
average perentage of vulnerabilities that are fixed
We consider the total numnber of vulnerabilities detected for a given host as per vulnerability scanner, calculate the ratio between those that are still present and those that are fixed and express this as a percentage.
